python3 composer.py 

compose 200*2 worlds each time by default. Change N if needed.

    worlds_static/
    worlds_dynamic/

worlds_static/static_seed_0001.world
worlds_dynamic/dynamic_seed_0001.world
...
worlds_static/static_seed_XXXX.world
worlds_dynamic/dynamic_seed_XXXX.world


Output:
    worlds_unified/unified_seed_0001.world
    ...
    unified_seed_XXXX.world
